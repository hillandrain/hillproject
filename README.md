# 猜颜色游戏

一个简单的颜色猜谜游戏 Android 应用。

## 游戏简介

- 根据提示猜测正确的颜色
- 挑战你的色彩识别能力
- 简洁的界面设计

## 技术栈

- Android (Java)
- GitHub Actions 自动构建

## 下载

可以从 [GitHub Actions](https://github.com/hillandrain/hillproject/actions) 页面下载最新构建的 APK。

## 项目结构

- `color-guess-apk-source/` - Android 项目源码
- `index.html` - 网页版游戏
- `.github/workflows/` - 自动构建配置

---

*Build with GitHub Actions*
